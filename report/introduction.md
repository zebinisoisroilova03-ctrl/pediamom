PediaMom is an AI-supported web application designed to assist parents
in monitoring and understanding their children's health data.
The system focuses on manual health data input, historical analysis,
and AI-assisted explanations while explicitly avoiding medical diagnosis.
The platform aims to improve health literacy, preventive awareness,
and responsible decision-making among parents.

In the first development phase, the Firebase project environment was set up, including authentication services, database initialization, and cloud hosting configuration. The application was successfully deployed using Firebase Hosting, enabling real-time testing and validation.

Authentication and authorization were implemented using Firebase Authentication. Access control was enforced through auth state listeners and Firestore security rules, ensuring that only authenticated parents can access and manage their own data.
